# Student-Performance-Analysis
An interactive Excel dashboard transforming raw student data into actionable performance insights using Pivot Tables, KPIs, and dynamic visualizations.
This project involved cleaning and transforming raw student data, creating calculated performance metrics, and building dynamic Pivot Tables and Pivot Charts to uncover trends. The dashboard features key performance indicators including average final score, pass rate, total students, and average absences, with interactive slicers that allow users to filter and explore the data dynamically.
